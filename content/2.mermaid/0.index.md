# Mermaid Diagrams and Charts

## Mermaid Component
[Component source code ðŸ’¾](https://github.com/losfroger/docus-docs-starter-mermaid/blob/main/components/content/Mermaid.vue)

To use mermaid diagrams and charts you can use the following syntax:
```
:mermaid{content='
# content here
'}
```
::alert{type="warning"}
You can use **single quotes** or **double quotes** for the content. But <u>**single quotes are recommended**</u> because some types of diagrams may use double quotes and it will cause an error.
::

Live reloading is supported, but using mermaid's [live editor âœ](https://mermaid.live/) is highly recommended.

---
